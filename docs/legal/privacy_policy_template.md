## 2. Какие данные мы собираем

2.1. При использовании Бота мы получаем следующие персональные данные:

| Категория данных | Описание | Источник |
|-----------------|----------|----------|
| Telegram ID | Уникальный идентификатор вашего аккаунта | Telegram API |
| Username | Имя пользователя (@username) | Telegram API |
| Имя | Имя и фамилия из профиля Telegram | Telegram API |
| Язык интерфейса | Код языка (ru, en) | Telegram API / выбор пользователя |

2.2. При оплате услуг мы дополнительно обрабатываем:

| Категория данных | Описание | Источник |
|-----------------|----------|----------|
| Токен платёжного средства | Идентификатор для повторных платежей | Платёжная система |
| Данные о транзакциях | Дата, сумма, статус платежа | Платёжная система |

2.3. **Мы НЕ собираем:**
- Номера банковских карт и другие финансовые реквизиты (обрабатываются только платёжной системой)
- Номер телефона или email (если вы не предоставляете их добровольно)
- Геолокацию или данные об устройстве

## 5. Хранение и защита данных

5.1. **Место хранения:** Персональные данные хранятся на серверах, расположенных на территории Российской Федерации, в соответствии с требованиями Федерального закона №152-ФЗ «О персональных данных».

5.2. **Меры защиты:**
- Доступ к данным ограничен и контролируется
- Базы данных защищены паролями и шифрованием
- Передача данных осуществляется по защищённым каналам (TLS/SSL)

5.3. **Сроки хранения:**
- Данные профиля (Telegram ID, username, имя) — на протяжении использования Бота и 1 год после последней активности
- Данные о платежах — 5 лет (требование налогового законодательства)
- После истечения сроков данные удаляются или обезличиваются
