# Инструкция по установке Python 3.11

## ✅ Скрипт запущен

Обновлённый скрипт `install_python311_direct.py` запущен и работает в фоновом режиме.

### Что происходит сейчас:

1. **Попытка скачать Python 3.11.9** (последняя стабильная версия)
2. Если не получится — попробует Python 3.11.8
3. Если и это не сработает — попробует Python 3.11.7

### Что вы увидите:

В окне терминала должно появиться:
```
[1/3] Скачивание установщика Python 3.11.11...
Пробую скачать Python 3.11.9...
Скачано: XX%
✅ Установщик скачан: C:\Users\...\python-3.11.9-amd64.exe
```

### После скачивания:

1. **Автоматически откроется окно установки Python**
2. **ВАЖНО:** В окне установки:
   - ✅ Отметьте галочку **"Add Python to PATH"** внизу окна
   - Нажмите **"Install Now"**
   - Дождитесь завершения установки
   - Нажмите **"Close"**

## Если скрипт не сработал

### Ручная установка (рекомендуется):

1. **Откройте в браузере:**
   https://www.python.org/downloads/

2. **Прокрутите вниз до Python 3.11.x**

3. **Скачайте "Windows installer (64-bit)"**

4. **Запустите скачанный файл**

5. **В окне установки:**
   - ✅ Отметьте **"Add Python to PATH"**
   - Нажмите **"Install Now"**

## После установки

### Шаг 1: Перезапустите командную строку
Закройте и откройте заново PowerShell или командную строку.

### Шаг 2: Проверьте установку
```bash
py -3.11 --version
```
Должно показать: `Python 3.11.x`

### Шаг 3: Создайте виртуальное окружение

**Автоматически:**
```bash
recreate_venv_python311.bat
```

**Или вручную:**
```bash
py -3.11 -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
pip install -r requirements-dev.txt
```

## Проверка работы

После создания виртуального окружения проверьте:
```bash
.venv\Scripts\python.exe --version
```
Должно показать: `Python 3.11.x`

## Альтернатива: Использовать Python 3.13

Если установка Python 3.11 вызывает проблемы, можно использовать уже установленный Python 3.13.9:

```bash
python -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
pip install -r requirements-dev.txt
```

Python 3.13.9 полностью совместим с проектом (требуется >= 3.11).
