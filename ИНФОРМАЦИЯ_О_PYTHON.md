## Как проверить версию самостоятельно

### Способ 1: Через командную строку
```bash
python --version
```

### Способ 2: Через Python
```bash
python -c "import sys; print(sys.version)"
```

### Способ 3: Запустить скрипт проверки
```bash
python check_python_version.py
```
